# คำถาม คำตอบ ตัวอย่าง
1.

CSS Syntax ประกอบด้วยอะไรบ้าง

 ประกอบด้วย selector และ declaration block โดย declaration block ประกอบด้วย property และ value
2. การเขียน CSS มีกี่แบบอะไรบ้าง มี 3 แบบ Extanal, Intanal, Inline 
                    1.Extanal จะเขียนไว้อีกไฟล์โดยที่เรียกใช้แบบนี้ 
                    <link rel="stylesheet" href="mystyle.css">

                    2.Intanal จะเขียนไว้ในหน้าเดียวกัน แต่จะใส่ tag style เช่น
                    <style>
                        body {
                          background-color: linen;
                        }
                        
                        h1 {
                          color: maroon;
                          margin-left: 40px;
                        }
                    </style>
3. Selector มีกี่แบบอะไรบ้าง มี 5 ประเภท คือ
  • Simple selectors จะ based on name, id, class
  • Combinator selectors
  • Pseudo-class selectors
  • Pseudo-elements selectors
  • Attribute selectors
 
                    Simple selectors จะ based on name, id, class

                    Combinator selectors จะมีการใช้สัญลักษณ์ ดังนี้
                        descendant selector (space)
                        child selector (>)
                        adjacent sibling selector (+)
                        general sibling selector (~)

                    Pseudo-class selectors จะมีการใช้ state ของ element เช่น 
                        /* mouse over link */
                            a:hover {
                            color: #FF00FF;
                        }

                    Pseudo-elements selectors จะใช้แค่ส่วนหนึ่งของ element เช่น
                        p::first-line {
                            color: #ff0000;
                            font-variant: small-caps;
                        }

                    Attribute selectors ใช้เฉพาะบาง attribute เช่น
                        a[target="_blank"] {
                            background-color: yellow;
                        }
4. Combinator มีกี่แบบอะไรบ้าง มี 4 แบบ คือ
  • descendant selector (space)
  • child selector (>)
  • adjacent sibling selector (+)
  • general sibling selector (~)
 
                descendant selector (space)
                    div p {
                        background-color: yellow;
                    }

                child selector (>)
                    div > p {
                        background-color: yellow;
                    }

                adjacent sibling selector (+)
                    div + p {
                        background-color: yellow;
                    }

                general sibling selector (~)
                    div ~ p {
                        background-color: yellow;
                    }
